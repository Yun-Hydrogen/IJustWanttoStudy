## 📖 简介 (Introduction)

**IJWS-I Just Want to Study-我只是想学习** 是一个为 **组卷网** 设计的 Tampermonkey脚本。

核心理念是 **"Less is More"**。通过移除页面上繁杂的干扰元素（标题、侧边栏、推荐），并将网页重构为类似试卷的布局，帮助你专注于题目本身。无论作为学生，想浏览题目核对答案或进行模拟考试，还是作为考试，预览试题内容,IJWS 都能提供最佳的视图支持。

## ✨ 核心功能 (Features)

### 1. 🛡️ 极简阅读 (Clean UI)
- **去噪**：自动隐藏页眉、页脚、侧边栏等干扰。
- **重排**：优化页面排版，调整为适合阅读的宽度和字体，去除组件。


### 2. 🔄 三大模式 (Three Modes)

通过页面右下角的 **悬浮球** 即可快速切换：

*   **📝 试卷模式 (Exam Mode)**
    *   仅显示题目，隐藏一切无关信息。
    *   适合纯粹的题目浏览，提供类似纸质试卷的阅读感。

*   **👀 参考模式 (Reference Mode)**
    *   在试卷模式基础上，**自动展开所有答案和解析**。
    *   适合做题后的复盘、核对和背诵。

*   **✍️ 练习模式 (Practice Mode)**
    *   **隐藏题目下方的答案区域**，将所有答案统一移动到页面 **最底部** 的答案区。
    *   **模拟练习**：先看着题目做，做完后滚动到底部统一对答案和查看解析，防止视线无意中扫到答案影响练习效果。

## 🛠️ 安装指南 (Installation)

1.  **安装管理器**：首先确保你的浏览器已安装 [Tampermonkey](https://www.tampermonkey.net/)插件。
2.  **安装脚本**：
      - [点击此处](https://raw.githubusercontent.com/Yun-Hydrogen/IJustWanttoStudy/refs/heads/main/IJWS.js)下载(或复制)js文件。
    - 在 Tampermonkey 中添加新脚本并粘贴代码。
    - 或者访问发布页面直接点击安装。
3.  **启用脚本**：打开组卷网试卷详情页，脚本将自动运行，右下角出现悬浮球即代表成功。若未成功运行，请检查是否开启脚本或Tampermonkey是否有权限加载用户脚本。

## 🎮 使用说明 (Usage)

1.  **悬浮球**：页面右下角会出现一个 `IJWS` 悬浮球。
2.  **展开菜单**：点击悬浮球，展开子菜单。
3.  **切换模式**：点击对应的子悬浮球切换模式（再次点击当前模式可退出，恢复网页原样）。
    - 📝 图标：试卷模式
    - 👀 图标：参考模式
    - ✍️ 图标：练习模式
4.  **拖拽移动**：按住悬浮球可将其拖拽至屏幕任意位置，避免遮挡题目。
